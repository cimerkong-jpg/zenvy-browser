# Workflow Rules

## Git
- Commit nhỏ, mỗi commit 1 tính năng hoặc 1 fix
- Message format: `feat:`, `fix:`, `style:`, `refactor:`
- Không push thẳng lên main, dùng branch

## Code
- Viết code trước, comment sau nếu logic phức tạp
- Không thêm tính năng ngoài scope đã chốt trong CLAUDE.md
- Test thủ công trên macOS trước khi nói xong

## Review
- Chạy `npm run build` trước khi commit
- Kiểm tra console không có lỗi đỏ
